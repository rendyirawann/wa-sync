Autentikasi

Semua endpoint butuh appkey (per akun) & authkey (per nomor). appkey Anda: {{ app_key }}. authkey tiap nomor ada di menu API Keys. Base URL: {{ api_base }}. Rate limit: 60 permintaan/menit per appkey.

POST/api/create-message

Kirim pesan teks (masuk antrian throttle anti-ban).

appkeyappkey akun
authkeyauthkey nomor pengirim
tonomor tujuan (62…)
messageisi pesan
sandboxopsional, "true" = uji tanpa kirim
curl -X POST {{ api_base }}/api/create-message \
  -d "appkey={{ app_key }}" -d "authkey=AUTHKEY" \
  -d "to=628123456789" -d "message=Halo dari API"
POST/api/create-message-media

Kirim media via URL. Param tambahan: media (URL), type (image/video/document), message (caption).

curl -X POST {{ api_base }}/api/create-message-media \
  -d "appkey={{ app_key }}" -d "authkey=AUTHKEY" -d "to=628123456789" \
  -d "media=https://contoh.com/gambar.jpg" -d "type=image" -d "message=Caption"
POST/api/send-location

Kirim lokasi. Param: to, lat, lng, name.

curl -X POST {{ api_base }}/api/send-location \
  -d "appkey={{ app_key }}" -d "authkey=AUTHKEY" -d "to=628123456789" \
  -d "lat=-6.2088" -d "lng=106.8456" -d "name=Kantor Jakarta"
GET/api/messages

Ambil riwayat pesan. Query: appkey, authkey, jid (opsional), limit (default 50, maks 200).

curl "{{ api_base }}/api/messages?appkey={{ app_key }}&authkey=AUTHKEY&limit=20"
GET/api/contacts

Daftar kontak (nama + tag). Query: appkey, authkey.

curl "{{ api_base }}/api/contacts?appkey={{ app_key }}&authkey=AUTHKEY"
GET/api/status

Status nomor + sisa kuota harian. Query: appkey, authkey.

curl "{{ api_base }}/api/status?appkey={{ app_key }}&authkey=AUTHKEY"

Format respons

Sukses: {"status":true, ...}. Gagal: {"status":false,"code":"...","message":"..."} dengan HTTP status sesuai (401 kunci salah, 403 kuota habis, 429 rate limit).
